# 📋 Requisitos do Sistema

## ✅ Requisitos Funcionais

RF01 - O usuário deve poder se cadastrar  
RF02 - O usuário deve poder fazer login  
RF03 - O usuário deve registrar hábitos diários (água, sono, exercícios)  
RF04 - O sistema deve permitir visualizar o histórico de hábitos  
RF05 - O sistema deve apresentar um dashboard com o progresso do usuário  

## ⚙️ Requisitos Não Funcionais

RNF01 - O sistema deve ser acessível via navegador  
RNF02 - O sistema deve ser responsivo  
RNF03 - O tempo de resposta deve ser inferior a 2 segundos  
RNF04 - Os dados devem ser armazenados com segurança  
RNF05 - O sistema deve garantir disponibilidade mínima durante o uso
