# 🌱 VitalTrack - Monitor de Hábitos Saudáveis

## 📌 Objetivo
Este projeto tem como objetivo desenvolver uma aplicação de software alinhada ao ODS 3 (Saúde e Bem-Estar), ajudando usuários a monitorar seus hábitos diários.

## ❗ Problema
Muitas pessoas não possuem controle sobre hábitos básicos de saúde, como ingestão de água, sono e prática de exercícios.

## 💡 Solução
Um sistema web com frontend e backend que permite:
- Registrar hábitos diários
- Visualizar progresso
- Receber alertas

## 🛠️ Tecnologias (previstas)
- Frontend: React
- Backend: C# .NET
- Banco de dados: SQL Server 

## 📊 Metodologia
O projeto será desenvolvido utilizando Scrum, com sprints quinzenais.
